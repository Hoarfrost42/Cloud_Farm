## Reset 规则

### 雨阶 reset

触发条件：

```text
weather >= getRainRankRequirement(state)
```

效果：

- `rainRanks += 1`
- 本轮资源清零。
- 本轮升级清零，但保留季风冲刺层升级 `monsoonPull`。
- 点击冷却清零。
- 保留云核、总云核、季风次数、永久升级。
- 如果有 `凝雨熟练`，保留 `云层注入 / 活力基流 / 天气增幅` 的至少 Lv.1。

当前没有批量雨阶。即使天气活力超过多个门槛，也只获得 1 雨阶。
雨阶 reset 可以在 10 雨阶后继续执行，且不会清掉已购买的季风牵引。

### 季风 reset

触发条件：

```text
weather >= monsoonTarget
rainRanks >= 10
windEye > 0
```

效果：

- `monsoonCycles += 1`
- 获得云核。
- 本轮资源和本轮升级清零。
- 清空季风牵引。
- 保留永久天赋。

云核获得：

```text
baseCoreGain = 3 + floor((weather / target) ^ 0.25)
rankBonus = floor(max(0, rainRanks - 10) / 2)
lensBonus = hasMonsoonLens ? 1 : 0
cloudCoreGain = clamp(baseCoreGain + rankBonus + lensBonus, 4, 12)
```

4. 自动化组已经有条目，但季风后尚未真正调成稳定中期平台。
